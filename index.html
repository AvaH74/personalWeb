Task Manager

Visit the website here: Task Manager

Task Manager is a full-stack web application built using React.js for the front end and Express.js with MongoDB and Mongoose for the back end. The application provides users with a simple and efficient way to manage their to-do lists and tasks. Users can register and log in securely using Passport.js for authentication.


Features:

  • User Registration and Login: Secure user authentication and registration system using Passport.js

  • To-Do List Management: Users can create, edit, and delete tasks in their to-do lists.

  • Real-time Updates: Updates to the to-do lists are instantly reflected for all users in real-time

  • Motivational Quotes: The app includes a feature to generate random motivational quotes to inspire users

  • Responsive Design: The application is responsive and works seamlessly on various devices and screen sizes
